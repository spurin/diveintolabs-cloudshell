# DiveInto Labs - Google Cloud Shell

![DiveInto](https://raw.githubusercontent.com/spurin/diveintolabs-cloudshell/main/logo.jpg)

This tutorial provides you with the DiveInto Labs, accessible in your browser 🚀

Launch the lab with the following (click the first icon above the command to send to terminal) -

```bash
curl -s https://raw.githubusercontent.com/spurin/diveintolabs-cloudshell/refs/heads/main/run.sh | bash
```

Once loaded, to access the Portal click the Web Preview Icon, if you cant find it, click -> <walkthrough-web-preview-icon>here</walkthrough-web-preview-icon> for a walkthrough on where to find it. Then select 'Preview on Port 8080' and you're good to go!
